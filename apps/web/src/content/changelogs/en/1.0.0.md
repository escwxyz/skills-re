After several months of private beta with a handful of authors, skills.re is now open to everyone. Any developer can browse the registry, install skills into their Claude environment, and publish new ones directly from a GitHub repository.

The initial registry ships with over 40 community-contributed skills covering code review, documentation generation, PR workflows, and more.
