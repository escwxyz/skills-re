---
question: 如何发布 Skill？
order: 2
---

发布 Skill 分三步：

1. **创建 `SKILL.md`** — 在 GitHub 仓库中添加所需 frontmatter（`name`、`description`、`license`、`compatibility`、`metadata`）。
2. **连接 GitHub 账户** — 在设置 → 集成 → GitHub 中授权 skills.re GitHub App。
3. **提交** — 前往 [skills.re/submit](/submit)，选择仓库和版本标签。

自动检查会立即运行。全部通过后，您的 Skill 将在几分钟内上线。完整教程请参阅[发布 Skill](/docs/publishing-skills)。
