---
question: How can I contribute to the registry?
order: 6
---

There are several ways to contribute:

**Publish skills** — the most direct contribution. Write a `SKILL.md`, test it thoroughly, and submit it. Quality skills with clear descriptions and accurate compatibility flags are always welcome.

**Report issues** — if a listed skill doesn't work as described on a supported runtime, open an issue on the skill's GitHub repository or flag it in the registry using the **Report** button on the skill page.

**Improve documentation** — the documentation source is on GitHub. Open a pull request to fix errors, add examples, or translate content into a new language.

**Spread the word** — share skills you find useful, write blog posts, or mention the registry in your community. Growing the ecosystem benefits everyone.

Join the conversation on Discord or follow the project on GitHub to stay up to date with new features and discussions.
