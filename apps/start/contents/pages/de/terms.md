## 1. Akzeptanz

Durch den Zugriff auf die skills.re-Registry, die Installation der CLI oder die Nutzung einer API stimmen Sie diesen Bedingungen zu.

## 2. Nutzung der Registry

Sie dürfen Skills ausschließlich für rechtmäßige Zwecke installieren und ausführen. Es ist untersagt, Skills zur Umgehung von Sicherheitskontrollen, zur Erstellung verbotener Inhalte oder zum Verstoß gegen geltendes Recht zu verwenden.

## 3. Veröffentlichung von Skills

Von Ihnen veröffentlichte Skills müssen den Einreichungsrichtlinien entsprechen. Sie behalten das Eigentum an Ihren Inhalten, gewähren skills.re jedoch eine Lizenz zur Verteilung über die Registry.

## 4. Haftung

Die Registry wird ohne Gewährleistung bereitgestellt. Wir übernehmen keine Garantie für die Richtigkeit oder Eignung von Skills. Die Nutzung erfolgt auf eigenes Risiko.

## 5. Änderungen

Wir können diese Bedingungen jederzeit aktualisieren. Die fortgesetzte Nutzung gilt als Zustimmung zu den geänderten Bedingungen.
