---
question: How do I publish a skill?
order: 2
---

Publishing a skill takes three steps:

1. **Create a `SKILL.md`** in a GitHub repository with the required frontmatter (`name`, `description`, `license`, `compatibility`, `metadata`).
2. **Connect your GitHub account** via Settings → Integrations → GitHub and authorise the skills.re GitHub App for that repository.
3. **Submit** at [skills.re/submit](/submit) — select the repository and the version tag to publish.

Automated checks run immediately. If everything passes, your skill goes live within a few minutes. See the [Publishing Skills](/docs/publishing-skills) guide for the full walkthrough.
