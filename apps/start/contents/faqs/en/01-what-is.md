---
question: What is the Agent Skill Registry?
order: 1
---

The Agent Skill Registry (skills.re) is a centralised repository for discovering, versioning, and distributing skills for AI agents. A skill is a structured markdown file (`SKILL.md`) that instructs an agent how to perform a specific task — like reviewing code, writing tests, or triaging issues.

Unlike general prompt libraries, skills are versioned, signed, and tested against a standard evaluation suite before being listed in the registry. This means you can trust that a skill behaves as described when you install it.
