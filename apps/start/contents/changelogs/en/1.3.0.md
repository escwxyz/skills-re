The GitHub Import flow replaces the manual upload process entirely for most authors. Point the importer at any public repository and it discovers all `skill.json` manifests, runs the eval suite, and queues the skills for review.

Webhook auto-import means your registry entry stays in sync with your repo without any manual intervention.
