Collections let authors like `@anthropic` ship a cohesive set of skills — for example, a "PR workflow" collection that includes code review, PR summary, and changelog generation as a single installable unit.

Verified author status is granted manually during the early period. We'll automate the criteria once the ecosystem grows.
